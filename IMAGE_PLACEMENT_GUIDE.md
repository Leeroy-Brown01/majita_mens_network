# Image Placement Guide

Please save the attached images to the following locations:

## 1. Logo
- **File**: The Majita logo image (with pink and blue hearts)
- **Save as**: `public/images/majita-logo.png`
- **Usage**: Top-left navbar logo

## 2. Team/Community Photos
Save the team group photos to:
- `public/images/team-group-1.jpg` (first team photo)
- `public/images/team-group-2.jpg` (second team photo)
- `public/images/activism-shirt.jpg` (photo showing "16 Days of Activism" t-shirt)
- `public/images/team-member-1.jpg` (individual portrait)

## How to Add Images
1. Navigate to `public/images/` folder in your project
2. Drag and drop or copy the images with the exact names above
3. The website will automatically load them
